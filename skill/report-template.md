# 每日动量报告 HTML 模板规范

## 报告结构

报告为单文件 HTML，使用内联 CSS，风格为深色金融终端风。

### 必需部分

1. **报告头部**
   - 标题：「土豆条子 每日动量报告」
   - 日期：当前交易日
   - 免责声明徽标

2. **市场立场板块**
   - 大号显示立场标签（三选一）：
     - 🟢 激进买入（Aggressive Buy）
     - 🟡 保守买入（Conservative Buy）
     - 🔴 持币观望（Hold / Cash）
   - 立场理由简述（2-3句）
   - 关键指标卡片：VIX/iVIX、上证指数、深证成指、股指期货（IF/IH/IC）

3. **5% 观察名单**
   - 表格形式，列：排名、股票代码、股票名称、胜率概率、选择理由
   - 胜率用进度条可视化
   - 每只股票理由控制在2-3句

4. **市场概览**
   - 热点板块（涨幅前5板块）
   - 主力资金流向摘要
   - 涨停/跌停统计

5. **风险免责声明**
   - 固定文案：「本报告仅供学习交流，不构成任何投资建议。股市有风险，入市需谨慎。土豆条子不对任何因使用本报告而产生的投资损失承担责任。所有胜率概率为主观判断，不代表实际收益保证。」

### 样式要求

- 深色背景（#0d1117），浅色文字
- 强调色：绿色(#00d97e)表示看涨，红色(#ff4d4f)表示看跌，黄色(#ffc107)表示中性
- 注意：A股用红涨绿跌配色（与美股相反），在涨跌幅数字显示时需注意
- 等宽字体用于数据显示
- 响应式布局
