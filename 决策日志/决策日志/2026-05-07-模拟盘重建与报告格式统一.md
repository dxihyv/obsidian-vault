## 3. 报告格式统一
**时间**：2026-05-07 傍晚
**决策**：所有定时任务推送格式对齐为统一风格

**统一规范**：
- 标题：`【N️⃣ 名称】` + `──────────────────────────────`
- 持仓：每只股票分行展示（名称 → 现价/今日涨跌 → 成本/浮盈/金额 → 止损目标）
- 颜色：`🔴` 跌 / `🟢` 涨 / `⚪` 震荡
- 数值：`+` 正数，`-` 负数，`¥` 单位，`%` 放最后

## 4. MEMORY 使用原则确立
**背景**：因信任 MEMORY 里的版本号 v3.0，导致总结时给出错误信息
**规则**：
- MEMORY 只存"静态规则/常识"

- 动态信息 → 存对应文件或 Obsidian 决策日志
