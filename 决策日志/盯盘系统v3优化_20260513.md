# 盯盘系统 v3.0 优化

**日期**：2026-05-13

## 新增功能

### 1. 换仓推荐机制
止盈/止损执行后，自动扫描 `today_candidates.csv` 推荐接回标的。

**过滤条件**：
- 剔除 今日涨幅 ≥ 9%（涨停/加速，不追）
- 剔除 现价 > 买入区间高点 × 1.005（超出合理买点）
- 剔除 综合评分 < 75
- 只保留现价在买入区间内的标的

**仓位规则**：
- 大盘涨幅 > +1% → 仓位减半 30%
- 大盘涨幅 ≤ +1% → 标准半仓 50%

**推送卡片包含**：代码、现价、买入区间、推荐仓位、止损价、目标价、筛选理由、触发原因

**函数**：`find_replacement_from_candidates()`、`send_reentry_alert()`
**冷却**：5分钟不重复推送

### 2. 真实持仓合并推送
多个条件同时触发，只推送最紧急的1条（优先级：硬止损 > 移动止损 > 超期 > 亏损警戒 > 信念预警）

### 3. 真实持仓超期持仓提醒
持仓 ≥ 15天推送提醒（之前仅模拟盘有）

### 4. 真实持仓信念分预警
- 信念 < 55 → 推送"信念清仓"预警
- 信念 < 60 → 推送"信念减仓"预警
与模拟盘同等风控
