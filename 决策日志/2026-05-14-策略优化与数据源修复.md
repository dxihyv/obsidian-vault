## 2. 新增策略B（突破型）—— morning_report.py

### 背景

早报原有策略C（埋伏型），缺少"已启动但还没加速"的票。策略B补上了两者的交界点——主力确认入场且方向已向上。

### 策略B条件

| 条件 | 阈值 | 说明 |
|------|------|------|
| 量比 | > 40 | 主力资金在场 |
| 突破MA20 | price > ma20 | **方向确认（新增关键条件）** |
| 涨幅 | 3% - 8% | 已启动但未加速 |
| MACD | 红柱 > 0 | 趋势向上 |

### 策略C vs 策略B

| | 策略C（🔍埋伏型） | 策略B（🚀突破型） |
|---|---|---|
| 定位 | 主力悄悄买，等爆发 | 已启动，顺势跟进 |
| 涨幅 | < 5% | 3% - 8% |
| MA20 | 不要求 | **必须突破** |
| 止损 | -4%（更严） | -3%（正常） |
| 买入区间 | ref×0.96 | ref×0.98（略宽） |
| 信号图标 | 🔍 | 🚀 |

### 代码改动

